# 🎯 Apresentação do Sistema - TechBlog

## 📖 Sobre o Projeto

**TechBlog** é uma plataforma moderna de blog técnico desenvolvida para escritores e desenvolvedores compartilharem conhecimento. Com interface elegante em dark mode, editor rico e sistema completo de gerenciamento, oferece uma experiência profissional tanto para autores quanto para leitores.

## 🎯 Fluxo de Uso Principal

### Para Novos Usuários

```
1. Acesse a Home Page
   ↓
2. Clique em "Cadastrar"
   ↓
3. Preencha os dados (Nome, Email, Senha)
   ↓
4. Faça Login com as credenciais
   ↓
5. Acesse o Dashboard
   ↓
6. Clique em "Novo Artigo"
   ↓
7. Crie seu primeiro artigo
   ↓
8. Publique e compartilhe!
```

### Para Autores

```
Dashboard → Visualizar Estatísticas
    ↓
Novo Artigo → Escrever Conteúdo
    ↓
Upload de Banner → Preview
    ↓
Adicionar Tags → Categorizar
    ↓
Publicar → Visualizar
    ↓
Editar Perfil → Personalizar
```

### Para Leitores

```
Home → Explorar Artigos
    ↓
Buscar/Filtrar → Encontrar Conteúdo
    ↓
Clicar no Artigo → Ler
    ↓
Comentar → Interagir
    ↓
Curtir → Engajar
```

## 📦 Como Reproduzir os GIFs

Se você quiser gravar seus próprios GIFs de demonstração:

### Ferramentas Recomendadas

**Windows:**
- [ScreenToGif](https://www.screentogif.com/) (Grátis)
- [ShareX](https://getsharex.com/) (Grátis)
- [LICEcap](https://www.cockos.com/licecap/) (Grátis)

**Mac:**
- [Kap](https://getkap.co/) (Grátis)
- [GIPHY Capture](https://giphy.com/apps/giphycapture) (Grátis)

**Web:**
- [CloudApp](https://www.getcloudapp.com/)
- [Loom](https://www.loom.com/)

### Dicas para Gravação

1. **Resolução:** 1280x720 ou 1920x1080
2. **FPS:** 15-30 fps (30 para animações suaves)
3. **Duração:** 5-15 segundos por ação
4. **Tamanho:** Comprima para < 5MB
5. **Zoom:** Use zoom para destacar áreas importantes
6. **Cursor:** Mostre o cursor nas interações

### Cenários para Gravar

#### 1. Criação de Artigo
```
1. Estar logado no Dashboard
2. Clicar em "Novo Artigo"
3. Preencher título e conteúdo
4. Fazer upload de banner
5. Adicionar categoria e tags
6. Clicar em "Publicar"
```

#### 2. Configurações de Perfil
```
1. Navegar para Configurações
2. Clicar na foto de perfil
3. Selecionar nova foto
4. Ver preview instantâneo
5. Preencher bio
6. Clicar em "Salvar Alterações"
```

#### 3. Dashboard
```
1. Fazer login
2. Visualizar métricas
3. Scrollar pela lista de artigos
4. Hover nos botões de ação
5. Clicar em "Editar" de um artigo
```

#### 4. Busca e Filtros
```
1. Ir para "Todos os Artigos"
2. Digitar na busca
3. Ver resultados em tempo real
4. Clicar em um filtro de categoria
5. Alternar visualização (grade/lista)
```

#### 5. Autenticação
```
1. Clicar em "Cadastrar"
2. Preencher formulário
3. Submeter
4. Ver mensagem de sucesso
5. Fazer login
6. Ser redirecionado ao Dashboard
```

#### 6. Responsividade
```
1. Abrir DevTools (F12)
2. Ativar modo responsivo
3. Testar em diferentes tamanhos
4. Mostrar menu mobile
5. Interagir com elementos
```

## 🎓 Casos de Uso

### Caso 1: Blogger Iniciante
**Persona:** João, desenvolvedor júnior que quer compartilhar aprendizados

**Jornada:**
1. Cria uma conta rapidamente
2. Personaliza seu perfil com foto
3. Escreve seu primeiro artigo sobre React
4. Adiciona uma imagem de capa
5. Publica e compartilha nas redes sociais

**Resultado:** Artigo publicado em menos de 10 minutos

---

### Caso 2: Escritor Técnico Profissional
**Persona:** Maria, tech writer com 50+ artigos

**Jornada:**
1. Faz login em sua conta
2. Acessa dashboard para ver métricas
3. Edita artigo antigo com novas informações
4. Organiza conteúdo com tags
5. Monitora engajamento (visualizações, curtidas)
