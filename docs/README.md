- [ ] Tamanho < 5MB

## 📝 Convenções de Nomenclatura

- Use kebab-case (letras minúsculas com hífens)
- Seja descritivo mas conciso
- Evite números no nome (use palavras)
- Exemplos: `criar-artigo.gif`, `dashboard.png`

## 🚀 Como Usar

1. Grave os GIFs seguindo o [GUIA-GIFS.md](../GUIA-GIFS.md)
2. Salve os arquivos nas respectivas pastas
3. Verifique se os links em [APRESENTACAO.md](../APRESENTACAO.md) funcionam
4. Commit e push para o GitHub

## 💾 Tamanhos de Arquivo

### Limites Recomendados:
- **GIFs:** < 5MB cada
- **Screenshots:** < 2MB cada
- **Total da pasta:** < 50MB

### Se Ultrapassar os Limites:
- Comprima GIFs em [gifcompressor.com](https://gifcompressor.com/)
- Otimize PNGs em [tinypng.com](https://tinypng.com/)
- Considere usar resolução menor (1280x720)
